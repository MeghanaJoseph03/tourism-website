# Tourism Website 🌍

A simple and interactive tourism website built using **HTML, CSS, and Bootstrap**.  
This project allows users to explore famous tourist destinations with images, descriptions, and detailed views.

## ✨ Features

- Beautiful tourism landing page
- Favourite places section
- Detailed view for tourist destinations
- Image carousel using Bootstrap
- Smooth navigation between sections

## 🏛️ Featured Places

### Taj Mahal
Explore the beauty and history of the Taj Mahal, one of India's most famous monuments.

### Mysore Palace
Discover the royal architecture and cultural heritage of Mysore Palace.

### Victoria Memorial
Experience the elegance and historical importance of Victoria Memorial.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- JavaScript
